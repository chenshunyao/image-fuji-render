# Fuji Film Filter — MVP 需求文档（v2）

## 一、产品概述

一个纯前端的 Fuji 胶片滤镜网站。用户上传照片，通过滑动栏选择不同的 Fuji 经典胶片滤镜，实时预览效果，满意后下载处理后的照片。部署在 Cloudflare Pages。

## 二、滤镜来源

基于开源项目 **[pyrocat101/Fujifilm-LUTs](https://github.com/pyrocat101/Fujifilm-LUTs)** — 从 Fujifilm 官方 F-Log2C 色彩科学生成的 3D LUT。

**10 款 Fuji 胶片滤镜：**

| # | 滤镜名 | 风格 |
|---|--------|------|
| 1 | **PROVIA** | 标准色彩，自然均衡 |
| 2 | **Velvia** | 高饱和鲜艳，风景利器 |
| 3 | **ASTIA** | 柔和色调，自然肤色 |
| 4 | **CLASSIC CHROME** | 低饱和复古，纪实感 |
| 5 | **CLASSIC Neg.** | 高对比度，独特色彩渲染 |
| 6 | **REALA ACE** | 自然色彩还原 |
| 7 | **PRO Neg.Std** | 柔和色彩，人像向 |
| 8 | **ETERNA** | 电影感，低饱和低对比 |
| 9 | **ETERNA BB** | 银盐漂白效果，去饱和高对比 |
| 10 | **ACROS** | 经典黑白，细腻颗粒 |

技术处理：将 ProPhoto RGB `.cube` LUT 转换为 sRGB 色彩空间的 PNG LUT 纹理，前端通过 WebGL 着色器实时应用。

## 三、核心功能（MVP）

### 1. 照片上传
- 支持上传多张照片（点击 或 拖拽）
- 支持 JPG / PNG / WebP
- 上传后以缩略图网格展示，点击某张进入编辑视图

### 2. 滤镜选择
- 底部横向滑动栏，展示 10 款滤镜
- 每个滤镜选项显示：滤镜名称 + 当前照片的小缩略图预览
- 点击选中后高亮当前滤镜

### 3. 实时预览
- 选中滤镜后，主视图立即显示应用效果（WebGL 渲染，毫秒级）
- 支持原图/滤镜效果对比（点击按住显示原图，松开显示滤镜效果）

### 4. 下载
- 「下载」按钮，保存当前滤镜处理后的全分辨率图片
- 文件名格式：`原文件名_滤镜名.jpg`

## 四、用户流程

```
打开网站 → 上传照片（支持多张）
    ↓
缩略图网格展示 → 点击一张进入编辑
    ↓
底部滑动栏浏览滤镜（带缩略预览）→ 点击选择
    ↓
主区域实时预览效果 → 按住可对比原图
    ↓
满意 → 点击下载 → 保存到本地
    ↓
返回网格 → 选另一张继续，或上传更多
```

## 五、技术栈

| 层 | 选型 | 理由 |
|---|---|---|
| 框架 | React + Vite | 轻量快速，纯前端 |
| 图像处理 | WebGL + 3D LUT 纹理 | GPU 加速，实时渲染大图无压力 |
| 滤镜数据 | `.cube` → PNG LUT 纹理（构建时转换）| 浏览器友好，加载快 |
| 样式 | Tailwind CSS | 快速响应式 UI |
| 部署 | Cloudflare Pages | 用户指定 |

## 六、页面结构

单页应用，两个视图状态：

- **网格视图**：上传区域 + 已上传照片缩略图网格
- **编辑视图**：大图预览 + 底部滤镜滑动栏 + 下载按钮 + 返回按钮

## 七、MVP 不包含

- 用户账号/登录
- 滤镜强度调节
- 批量下载
- 后端服务

**技术方案变更：**
- 原计划使用 pyrocat101/Fujifilm-LUTs 的 .cube LUT 文件
- 实际采用参数化 WebGL fragment shader 方案：每款滤镜定义色调曲线、饱和度、对比度、色温、颗粒、暗角等参数，shader 统一渲染
- 原因：更轻量（无需加载外部 LUT 文件）、易于调参、浏览器兼容性好
